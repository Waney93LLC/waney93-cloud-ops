# waney93-cloud-ops
waney93-cloud-ops encodes enterprise cloud runbooks into secure, repeatable automation workflows, reducing operational risk, improving consistency, and accelerating delivery.

The framework demonstrates structured cloud governance through a Process → Orchestrator → Service model, supporting infrastructure management, application operations, and environment lifecycle control.

# Instructions
1. Clone the repository and navigate to the project directory.
2. Install dependencies using `npm install`.
3. To run the bootstrap process, use: `npm run dev -- <config-file>.json`. Ensure that the `<config-file>.json` file contains the correct AWS credentials and configuration for your environment.
